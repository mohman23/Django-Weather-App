About this app

This app has been created to check the weather forecast of Bengaluru, the weather of cities around the world and also to check the air qaulity of Bengaluru.

This app can be divided into three parts -

1. Home page which shows the weather forecast of Bengaluru. Data is being pulled from an api which belongs to Darksky. For more information about DarkSky please click here - Dark Sky.

2. The Weather page allows you to add cities to check their weather, it also allows you to delete cities from the page. This is CRUD(Create, Read, Update and Delete) enabled. Data is being pulled from an api which belongs to Open Weather. For more information about Open Weather please click here - OpenWeather.

3. Air Quality, this page allows you to check the air quality of three popular places within Bengaluru. We could have pulled this data from an api but instead we decided to scrap data from aqicn.org. For more information about aqicn.org please click here - AQICN.ORG.
